Plots
  • Formatting:
    • cex.main: Increase font size of chart title. Default is 1.
    • cex.axis: Increase font size of axes tick lables. Default is 1.
    • cex.lab: Increase font size of axes labels. Default is 1.
    • mar: Set margin size with a vector of length 4. First value specifies bottom margin, and move clockwise (to the left). Default setting is c(4.1, 5.1, 5.1, 2.1), which can be checked with par("mar").
    • oma
    • mfrow: Enable multiple plots in the same window with a vector of the form c(nrows,ncols). Default is c(1,1).
    • xpd: Enable plotting outside plot region, useful for when you want to put the legend outside the bounded box. Can toggle between TRUE and FALSE when you wish. Default value is FALSE, so all plots are restricted to plot region.
  • Legend:
    • legend()
    • bty: Toggle outline of legend on ("y") and off ("n"). Default is on.
    • ncol: Force horizontal legend by specifying number of desired columns. Default number of columns is 1.
    • cex: Increase font size.
    • title: Add a title to the top of the legend.
